Input to the Buoyancy (BUY) Package is read from the file that has type ``BUY6'' in the Name File.  If the BUY Package is not included for a model, then the model will use the constant density form of Darcy's Law for all flow calculations.  Only one BUY Package can be specified for a GWF model.
